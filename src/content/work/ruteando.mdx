      Ruteando
    </h3>
    <h2 class="workDetail__title">Enhancing the experience for bicycle users through an app</h2>
    <p class="workDetail__description">This application will allow you to navigate the city streets quickly and safely, providing real-time information on the most efficient routes and the safest places to ride. As part of this project, I had the opportunity to design the user interface and user experience of the application, focusing on creating a visually appealing and user-friendly experience.</p>

<section class="workPost__section workCase">
  <article class="workCase__item">
    <h3 class="workCase__title">Insight</h3>
    <ul class="workCase__list">
     <li class="paragraph">In recent years, the number of bike users in Bogotá has significantly increased, giving rise to new issues related to safety and accidents</li>
    </ul>
  </article>
  <article class="workCase__item">
    <h3 class="workCase__title">Solution</h3>
    <ul class="workCase__list">
     <li class="paragraph">An application that allows you to estimate arrival time based on user experience, filter routes based on speed or safety according to bike user preferences, and provide options to share the live journey with a family member. All of this with the aim of enhancing safety. Additionally, there is an option to save the completed route for future reference.</li>
    </ul>
  </article>
</section>
